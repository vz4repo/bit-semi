### 3. 계획후기
> 해당 계획의 후기
> 여행 사진 최소 1장 업로드 필수. 5장까지 업로드 가능
> 후기를 남겨야 해당 계획에 별점 등록 가능
```
후기 추천
후기 비추천
여행 계획 별점
```
```
후기 수정 기능
후기 삭제 기능
```
